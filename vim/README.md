# Vim Color Scheme - Blizzard Colors

In order to install when using [vim-plug](https://github.com/junegunn/vim-plug):

```vim
Plug 'kelsin/blizzard-colors', { 'rtp': 'vim' }
```

Then some point later on in your vimrc:

```vim
colorscheme blizzard
```
